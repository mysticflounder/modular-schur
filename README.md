# modular-schur

Supplementary material for **"The stabilized value of modular Schur numbers $S_m(k,\ell)$"**.

## What's here

- `paper/` — LaTeX source and compiled PDF.
- `lean/` — Lean 4 formalization (Aristotle-verified, axioms: `propext`, `Classical.choice`, `Quot.sound`).
- `scripts/` — Python SAT encoder (`schur_mod.py`) and stable-regime table generator (`phase9_stable_tables.py`).
- `docs/phase7-unified-theorem.md` — informal proof companion the paper abstracts from.

## Main theorem

For all $m \ge 2$, $\ell \ge 2$, and $k \ge m/\gcd(m,\ell-1) - 1$,
$$S_m(k,\ell) = \frac{m}{\gcd(m,\ell-1)} - 1.$$

Machine-verified headline in `lean/ModularSchur/Partition.lean`:

```lean
theorem schurModResidue_eq (m k ℓ : ℕ) (hm : 2 ≤ m) (hℓ : 2 ≤ ℓ)
    (hk : m / Nat.gcd m (ℓ - 1) - 1 ≤ k) :
    schurModResidue m k ℓ = m / Nat.gcd m (ℓ - 1) - 1
```

## Building

**Paper:** `cd paper && pdflatex modular-schur.tex`.

**Lean:** requires Lean 4 toolchain `leanprover/lean4:v4.28.0` and Mathlib hash `8f9d9cff6bd728b17a24e163c9402775d9e6a365` (pinned in `lean-toolchain` and `lake-manifest.json`).
```
cd lean && lake build
```

**Scripts:** require `uv`, CaDiCaL, and (optionally) `drat-trim` for certificate verification.
```
uv run scripts/phase9_stable_tables.py --help
```

## Reproducibility of SAT tables

The paper's *main theorem* is an elementary proof and does not depend on SAT. The empirical tables in `docs/phase7-unified-theorem.md` Appendix were generated by `scripts/schur_mod.py` + CaDiCaL, with DRAT certificates verified via drat-trim. Regenerating any single row takes seconds to minutes for small $m$. The full certificate tree occupies ~121 GB and is not distributed; `schur_mod.py` regenerates it deterministically.
